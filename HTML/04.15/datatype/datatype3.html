데이터 타입의 대전제

데이터 타입에 대해서 알아보기 전에, 우선 기본적이지만 굉장히 중요한 부분 하나만 짚고 넘어가보겠습니다. 바로 데이터 타입의 종류입니다.
데이터 타입에는 "원시 타입"과 "객체 타입"입니다. 이는 굉장히 중요한 부분이니 꼭 한번 짚어보고 넘어가겠습니다. 다만, 각 타입이 사용하는 언어에 따라 정의되는 부분이 다르기 때문에, 이번 문단에서는 자바스크립트 기준으로 설명 드리도록 하겠습니다.

원시타입

원시 타입(Primitive Type)은 총 6개의 종류로 구성되어 있습니다.

숫자 타입 문자열 타입 불리언 타입 undifined타입 null타입 symbol타입

객체 (참조)타입

객체 타입은 3종류로 구성이 되어있습니다.

객체 타입 배열 타입 함수 타입
